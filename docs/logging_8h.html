Detailed Description

提供带颜色和级别的日志记录功能。

包含 Logging 类，支持 DEBUG、INFO、WARN、ERROR、FATAL 五个级别， 可控制输出级别、颜色开关，并自动附加时间戳。全局默认 logger 实例可直接使用。
