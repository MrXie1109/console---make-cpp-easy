Detailed Description

template<class T>
class console::cursor_ptr< T[]>

双指针游标智能指针（数组版本）。

Template Parameters
T数组元素类型。

与非数组版本的区别在于析构使用 delete[]，赋值时使用 delete[] 释放原内存。

Warning
需确保与 new[] 匹配。
