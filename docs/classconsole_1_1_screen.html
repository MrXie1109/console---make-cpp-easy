Detailed Description

template<unsigned COLS, unsigned ROWS>
class console::Screen< COLS, ROWS >

用于表示虚拟屏幕的类。

使用八点位位盲文，每像素占用一比特位存储空间。

Template Parameters
COLS列数，必须是 2 的倍数。
ROWS行数，必须是 4 的倍数。
