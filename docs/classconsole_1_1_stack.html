Detailed Description

template<class T, class Container = std::deque<T>>
class console::Stack< T, Container >

栈容器适配器，扩展自 std::stack。

Template Parameters
T栈中存储的元素类型。
Container底层容器类型，默认为 std::deque<T>。

该适配器继承自 std::stack，提供了返回值版本的 pop() 方法。 使用示例：

console::Stack<int> s;
s.push(1);
s.push(2);
int value = s.pop(); // 返回 2 并弹出

弹出栈顶元素并返回其值。

Returns
typename base_type::value_type 栈顶元素的值（通过移动语义返回）。

该方法结合了 top() 和 pop() 操作，返回栈顶元素的副本/移动值， 然后将栈顶元素弹出。使用移动语义避免不必要的拷贝。
