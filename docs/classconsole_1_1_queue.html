Detailed Description

template<class T, class Container = std::deque<T>>
class console::Queue< T, Container >

队列容器适配器，扩展自 std::queue。

Template Parameters
T队列中存储的元素类型。
Container底层容器类型，默认为 std::deque<T>。

该适配器继承自 std::queue，提供了返回值版本的 pop() 方法。 使用示例：

console::Queue<int> q;
q.push(1);
q.push(2);
int value = q.pop(); // 返回 1 并弹出

弹出队首元素并返回其值。

Returns
typename base_type::value_type 队首元素的值（通过移动语义返回）。

该方法结合了 front() 和 pop() 操作，返回队首元素的副本/移动值， 然后将队首元素弹出。使用移动语义避免不必要的拷贝。
