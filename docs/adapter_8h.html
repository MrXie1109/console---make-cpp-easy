Detailed Description

提供带有值返回 pop 操作的容器适配器包装类和 range 函数。

该模块对 STL 容器适配器（stack、queue、priority_queue）进行扩展， 重载了 pop() 方法，使其在弹出元素的同时返回该元素的值。适用于需要在弹出元素时获取其值的场景。 提供 range() 函数用于生成数值范围，类似于 Python 的 range() 函数。
