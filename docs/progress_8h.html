Detailed Description

提供非侵入式的容器遍历进度条显示。

该模块允许你在遍历任何容器（或支持迭代器的范围）时，自动在控制台输出进度条。 进度条样式可通过 ProgressConfig 自定义，也提供了几种预定义样式。
