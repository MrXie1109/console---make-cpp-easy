Detailed Description

template<class T>
class console::Comprehension< T >

函数式风格的数据流处理容器。

Template Parameters
T容器中元素的类型。

该类包装一个 std::vector<T>，提供 map、filter 等惰性求值风格的操作（实际是立即求值）， 并支持从多种来源构造，如容器、迭代器对、初始化列表等。

Note
所有转换操作都会创建新的 Comprehension 对象，不会修改原对象。
