Detailed Description

提供 MultiArray 多维数组的数学工具函数。

包含统计（均值、方差、标准差）、线性代数（点积、范数、余弦、矩阵乘法、转置、迹、叉积）、 元素级运算（clamp、abs、三角函数、指数对数、幂、取整）、随机初始化、卷积等。
