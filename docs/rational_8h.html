Detailed Description

提供运行时的分数运算。

包含模板类 BasicRational 和数个预设的别名。 几乎所有操作都有可能导致整数溢出，请使用较大的整数类型或自行检查。 为了数值计算的效率，BasicRational 内部不做检查。
