Detailed Description

文件路径封装类，提供便捷的文件读写和路径操作。

在 Windows 平台上自动将 '/' 转换为 '\'，支持路径拼接运算符 /。 所有文件读写操作均会抛出 file_error 异常（若文件无法打开或流状态异常）。
