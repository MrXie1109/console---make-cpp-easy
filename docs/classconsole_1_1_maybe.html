Detailed Description

template<class T>
class console::Maybe< T >

可选值容器，可包含一个值或为空。

Template Parameters
T存储的值类型。

基于 std::unique_ptr 实现，支持拷贝、移动、流输入输出。 访问空 Maybe 会抛出 bad_maybe_access 异常。
