Detailed Description

提供 STL 容器的格式化输出和灵活的输出控制工具。

该头文件重载了 operator<< 使得标准容器（vector, list, set, map 等）、 pair 和 tuple 可以直接输出到流。同时提供了 Output 类用于灵活控制输出格式， 以及辅助函数 to_array 和 to_vector 用于将 C 风格数组转换为 STL 容器。
