Detailed Description

提供时间度量、计时、休眠和日期时间格式化功能。

包含 Time 类（纳秒精度时间量），支持单位转换、算术运算和比较；提供 now() 获取当前时间点， timer() 测量函数执行时间，sleep() 休眠，datetime() 获取格式化日期时间字符串。
