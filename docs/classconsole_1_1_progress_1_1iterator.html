Detailed Description

template<class Iter>
class console::Progress< Iter >::iterator

进度条的迭代器，负责绘制进度条。

该迭代器包装了底层迭代器，并在递增时根据已完成比例刷新进度条显示。 绘制频率限制为每 50 毫秒一次，以避免刷新过快。
