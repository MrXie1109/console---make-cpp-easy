Detailed Description

template<class T>
class console::RangeIterator< T >

RangeIterator 类模板，用于实现 range() 函数生成的迭代器。

该迭代器支持前置和后置递增操作，解引用操作返回当前值。 比较操作符根据步长方向判断是否达到范围边界。 可以直接用于 range-based for 循环中，生成数值范围。
