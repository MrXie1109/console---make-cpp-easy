Detailed Description

命令执行结果封装类。

存储命令执行状态码和标准输出内容，对象只能通过 syscmd 函数创建。

Returns
CmdResult 包含执行结果的对象。

自动合并 stderr 到 stdout，使用 256 字节缓冲区读取输出。

auto result = syscmd("ls", {"-la", "/tmp"}, {});
if (result.ok()) {
std::cout << result.output();
}
