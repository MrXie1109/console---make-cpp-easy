◆ default_gen()

std::mt19937 & console::default_gen ( )
inline

获取一个全局的、以当前时间戳为种子的 Mersenne Twister 随机数引擎。

该函数返回一个静态的 std::mt19937 对象的引用，引擎使用高精度时钟的当前时间点进行初始化。

Returns
std::mt19937& 随机数引擎的引用。

Parameters
