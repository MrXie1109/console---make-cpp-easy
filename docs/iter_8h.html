Detailed Description

提供些许关于迭代器的拓展功能。

本文件提供了两个主要工具：IteratorPair 用于将迭代器对包装为可范围遍历的对象， 以及 zip 函数族用于同时遍历两个容器，并在较短者结束时自动停止。
