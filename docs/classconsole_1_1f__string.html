Detailed Description

格式化字符串类，支持使用 % 运算符进行占位符 {} 替换。

继承自 std::string，通过 operator% 将第一个 {} 替换为参数的字符串表示。 若字符串中不包含 {} 则抛出 bad_format 异常。

使用示例：

f_string fmt = "Hello, {}!"_f;
std::string result = fmt % "world"; // "Hello, world!"
