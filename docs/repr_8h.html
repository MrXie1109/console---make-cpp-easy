Detailed Description

提供类似 Python 的 repr() 函数，用于生成对象的可读字符串表示。

该头文件通过 SFINAE 技术重载 repr 函数，支持字符串、字符、布尔值、nullptr、函数指针、 可打印类型（有 operator<<）以及不可打印类型（输出类型信息及地址）。
