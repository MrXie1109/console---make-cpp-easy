Detailed Description

template<class Container>
class console::View< Container >

容器的非拥有视图（可变版本）。

Template Parameters
Container容器类型，必须支持迭代器。

View 不拥有数据，仅保存一对迭代器，可以高效地表示容器的一个连续子区间。 支持迭代器、下标访问、边界检查的 at() 方法，以及 collect() 复制数据到新容器。

Note
需要确保底层容器在视图生命周期内有效。
