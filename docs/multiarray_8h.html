Detailed Description

提供编译期维度固定的多维数组容器 MultiArray，支持任意维度和元素级算术/逻辑/位运算。

MultiArray 继承自 std::array 的递归嵌套，支持 operator() 多维下标访问、扁平化迭代器、 元素级运算（包括数组与数组、数组与标量）、统计函数（sum/min/max/all/any）等。
