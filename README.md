# console.hpp v1.6.0: Logging and Time Utilities

## 📋 接口总览

### 🎯 核心输出系统
**`print` / `error`** - Python风格输出对象
- 可变参数，自动分隔
- 支持彩色输出：`print(Color::Red, "文本")`
- 可自定义分隔符和结束符

**`Output` 类** - 可配置输出器
- 构造函数设置流、结束符、分隔符
- `set()` 方法动态修改配置
- 支持颜色参数优先级处理

### 📥 智能输入系统
**`input()`** - 读取整行字符串
- 带提示符，简洁易用
- 自动处理缓冲区

**`input<T>()`** - 类型安全输入
- 模板参数指定返回类型
- 自动类型转换，清理缓冲区
- 支持所有 `operator>>` 可读类型

### 🎨 颜色系统
**`Color` 枚举** - ANSI颜色码
- 16种标准终端颜色
- 前景色（30-37）和亮色（90-97）
- `Reset` 重置终端颜色

**`colored()`** - 颜色代码生成
- 将 `Color` 枚举转为ANSI转义序列
- 自动集成到输出系统中

### ⏱️ 时间工具集
**`TimerResult` 结构** - 时间计量
- 纳秒/微秒/毫秒/秒四级存储
- 智能单位选择输出
- 支持算术运算

**`now()`** - 高精度当前时间
- 返回 `TimerResult` 对象
- 支持时间差计算

**`timer()`** - 函数执行计时
- 接受函数和参数，返回执行时间
- 简洁的性能测试工具

**`sleep()`** - 线程休眠
- 秒为单位，支持小数
- 简化 `std::this_thread::sleep_for`

**`datetime()`** - 格式化时间戳 **【v1.6.0新增】**
- 自定义格式字符串
- 包含毫秒精度
- 线程安全的时间转换

### 📊 日志系统 **【v1.6.0新增】**
**`Logging` 类** - 分级日志记录器
- 五个日志级别：DEBUG、INFO、WARN、ERROR、FATAL
- 自动时间戳和级别标签
- 彩色输出区分级别

**日志级别控制**
- `set(Level)` - 设置最低显示级别
- `set(bool...)` - 精细控制各级别开关
- 默认显示INFO及以上级别

**快捷方法**
- `debug()` - 亮黑色，调试信息
- `info()` - 亮青色，常规信息
- `warn()` - 亮黄色，警告信息
- `error()` - 亮红色，错误信息
- `fatal()` - 亮品红，致命错误

### 🔧 字符串工具
**修剪函数**
- `trim()` - 去除两端空格
- `ltrim()` - 去除左侧空格
- `rtrim()` - 去除右侧空格

**大小写转换**
- `upper()` - 全转大写
- `lower()` - 全转小写
- `title()` - 标题化（首字母大写）

**分割与组合**
- `split()` - 按分隔符分割字符串
- `join()` - 用分隔符连接字符串向量
- `partition()` - 三部分分割，返回结构体

### 🎲 随机工具
**`randomint()`** - 整数随机数
- 指定范围 [min, max]
- 均匀分布

**`uniform()`** - 浮点随机数
- 指定范围 [min, max]
- 均匀分布

**`choice()`** - 容器随机选择
- 从任意容器随机选择一个元素
- 支持 vector、array、set 等

### 📈 进度显示
**`ProgressBar` 类** - 可配置进度条
- 自定义边界字符、填充字符
- 可调长度，百分比显示
- 自动范围限制

**`jdt()`** - 快速进度条 **【拼音彩蛋】**
- 固定样式，简洁接口
- 回车符刷新，适合循环显示
- "进度条"拼音缩写

### ⏳ 等待动画
**`wait()` 三重载**
1. `wait(int)` - 固定次数旋转
2. `wait(bool)` - 条件变量控制
3. `wait(F)` - 函数条件控制
- 旋转动画反馈
- 可自定义结束消息

### 💾 文件工具
**`save()` / `load()`** - POD类型序列化
- 二进制保存/加载
- 简单错误处理
- 仅限平凡可复制类型

### 🖥️ 容器美化输出
**运算符重载** - 美观打印
- `vector<T>` - 方括号格式 `[1, 2, 3]`
- `array<T, n>` - 同 vector 格式
- `pair<T, U>` - 圆括号格式 `(first, second)`
- `map<K, V>` - 花括号键值对 `{key: value}`
- `set<T>` - 花括号集合 `{1, 2, 3}`
